# ZELDA

ZELDA is a premium men's underwear e-commerce application. Designed to deliver high-quality shopping experiences across Mobile, Web, and Desktop.

## Features
- **Modern Storefront:** Beautiful, responsive grid showcasing the newest and trending collections.
- **Product Details:** Rich product views with size and color selection.
- **Cart Management:** Fully functional local shopping cart (add, remove, adjust quantities).
- **Responsive Layout:** Adaptive user interface built seamlessly for both mobile devices and wide desktop displays.

## Tech Stack
- Flutter (Cross-platform UI)
- State Management (Built-in `ChangeNotifier` / `InheritedNotifier`)
- Responsive Design (LayoutBuilder & MediaQuery)

## Setup
1. Ensure Flutter is installed and configured on your machine.
2. Run `flutter pub get` to fetch any dependencies.
3. Start the application with `flutter run`.
